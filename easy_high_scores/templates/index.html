easy to use

I like to compete in game jams and create prototypes. Often, I'll want to add online high scores to my projects but the current free solutions are either too bloated and require a lot of setup, or they're very restricted, e.g., HTTPS not allowed (which affects some web engines), or the amount of scores that can be stored is severely limited. My goal with this project was to create a solution for developers that is easy to use and simple to integrate.

  • Open source
  • Beginner friendly
  • One-click instant setup
  • RESTful API (GET/POST/DELETE)
  • A shareable public code, allowing read-only access
  • JSON-based, compatible with the latest standard
  • An additional simple API relying solely on GET requests
  • Submit scores and get updated score-list with via one request
  • Return top x scores (sorted by numerical data, scores may contain letters)
  • Capped at 2000 scores per user, after which the lowest scores get bumped

the basics

Here are some of the most important routes. You'll want to check out the documentation for further information such as HTTP status codes.

Add Scores
POST /api/<your private key>
