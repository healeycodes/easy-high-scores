easy to use

I like to compete in game jams. Often, I'll want to incorporate a leaderboard but the current free options are either too bloated and required a lot of setup, or they're very restricted, e.g., HTTPS not allowed (which affects some web engines), or the amount of scores that can be stored is severely limited. My goal with this project was to create a solution for developers that is easy to use and simple to integrate.

  • Open source
  • One-click instant setup
  • RESTful API (GET/POST/DELETE)
  • JSON-based, compatible with the latest standard
  • An additional simple API relying solely on GET requests
  • Submit scores and get updated score-list with via one request
  • Return top x scores (sorted by numerical data, scores may contain letters)
  • Capped at 2000 scores per user, after which the lowest scores get bumped
