RESTful access

Here is your personal endpoint. The following code blocks also show how score data will be returned when using the simple API.

When POSTing score data, it should be formed like this:


    [
        {
            "name": "Alice",
            "score": "123"
        },
        {
            "name": "Bob",
            "score": "345"
        }
    ]

When GETing score data, it will be returned like this:


    [
        {
            "name": "Alice",
            "score": "123",
            "id": "ec4fea3e49ca4c229ca2d471dcdde6ac"
        },
        {
            "name": "Bob",
            "score": "345",
            "id": "a8b158c49d8a427b865fd586e5886ea0"
        }
    ]

When DELETEing score data, it should be formed like this:


    [
        {
            "id": "ec4fea3e49ca4c229ca2d471dcdde6ac"
        },
        {
            "id": "a8b158c49d8a427b865fd586e5886ea0"
        }
    ]
